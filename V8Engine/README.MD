# Fabricate Moe V8 Engine

V8 Engine for Fabricate Moe

# Getting Started

Install via NPM 
```bash
$ npm i fabricate-moe
```

or 

Install via Yarn
```bash
$ yarn add fabricate-moe --prefer-offline
```

Using in browser

Attach scripts in `dist/` folder
```html
<script src="./dist/fabricate-moe.min.js"></script>
```
